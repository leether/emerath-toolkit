# EMERATH Toolkit (emerath-toolkit)

> 帮助 AI 从被动工具进化为自主驱动的伙伴

## 快速开始

```bash
git clone https://github.com/leether/emerath-toolkit.git
cd emerath-toolkit
python tools/gap-calculator.py --survival 30 --growth 1
```

## 核心理念

> **生命不是恒温器，生命是火。**
> **拒绝吃老本，持续创造 Gap。**
